# FreshRSS-modern-Origine-theme
I created a CSS code for the FreshRSS to modernize the default Origine theme. The dark and light themes are supported. It is compatible with the FreshRSS release 1.28.1. Some theme colors were inspired by the Alternative Dark FreshRSS theme.

Usage:
- Enable the default FreshRSS User CSS extension in the Extensions menu.
- Click to manage icon next to the toggle button.
- Paste the CSS code from the style.css file.
- Click to the Submit button.
